# Calculator Pro

프리미엄 계산기 서비스로 일상의 모든 계산을 더 스마트하게 처리하세요.

## 서비스 소개

Calculator Pro는 iOS의 세련된 디자인을 기반으로 한 고급 웹 계산기입니다. 
단순한 계산 도구를 넘어 개인의 계산 히스토리와 고급 기능을 제공하는 프리미엄 구독 서비스입니다.

## 핵심 기능

**무료 기능**
- 실시간 계산 미리보기
- 백분율 계산
- 부호 변경 기능

**프리미엄 기능 (구독 후 이용 가능)**
- 계산 히스토리 자동 저장
- 고급 수학 함수 (사칙연산)
- 개인화된 설정
- 광고 없는 환경

## 시작하기

1. 웹브라우저에서 Calculator Pro에 접속
2. 평소와 같이 계산 수행
3. 등호(=) 버튼을 눌러 구독 시작
4. 원하는 플랜을 선택하고 결제 완료
5. 프리미엄 기능을 즉시 이용 시작
